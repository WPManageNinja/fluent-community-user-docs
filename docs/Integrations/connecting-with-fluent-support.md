# Connecting With Fluent Support

Integrating FluentCommunity with **Fluent Support** creates a more efficient and context-rich support experience. When connected, your support agents can see a customer's community and course details directly within the ticket view, helping them resolve issues faster.

> **Use Case:** A student submits a support ticket saying they are having trouble with "Lesson 3." Without leaving the ticket, your support agent can instantly see a **FluentCommunity Widget** that confirms which course the student is enrolled in, how far they have progressed, and other relevant details, providing immediate context for the issue.

## How the Integration Works

This integration adds two key features to your Fluent Support portal: a custom field for ticket submission and a widget in the agent's ticket view.

### 1. Custom Field for Course Selection

When the integration is active, you can add a "FluentCommunity Courses" custom field to your support ticket submission form.

1.  From your WordPress dashboard, navigate to **Fluent Support → Custom Fields**.
2.  Create a new field and select **Fluent Community Courses** as the field type.
3.  When customers submit a ticket, they can now select the specific course their issue is related to from a dropdown menu.

[Screenshot: The Fluent Support custom field settings, showing the "Fluent Community Courses" field type being selected.]

This allows you to create workflows in Fluent Support to automatically route tickets about a specific course to a specialized agent or department.

### 2. FluentCommunity Widget in Tickets

The most powerful feature is the widget that appears in the agent's ticket view. When an agent opens a ticket from a customer, a **FluentCommunity Widget** will be displayed in the sidebar.

This widget shows key information about the customer's status, including:
* **Enrolled Courses:** A list of all courses the user is enrolled in.
* **Enrollment Date:** When the user joined each course.
* **Course Progress:** The percentage of the course the user has completed.

[Screenshot: The agent's ticket view in Fluent Support, with the FluentCommunity Widget highlighted in the sidebar, showing a customer's enrolled courses and progress.]

This gives your support team instant access to the information they need without having to manually look up the user in your community, saving time and leading to faster resolutions.
