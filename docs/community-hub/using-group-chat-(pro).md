# Using Group Chat (Pro)

The **Group Chat** feature allows you to add a real-time chat room directly inside any Space. This is a powerful tool for fostering instant communication, hosting live discussions, or providing a more informal way for members to connect.

> **Use Case:** If you are running a live webinar or a virtual event, you can create a dedicated Space for it and enable the Group Chat. This gives attendees a "side-channel" to chat with each other in real-time while the event is happening.

## Enabling Group Chat for a Space

Group Chat is a feature that you enable on a per-space basis.

1.  Navigate to **FluentCommunity → Spaces** from your WordPress dashboard.
2.  Create a new Space or **Edit** an existing one.
3.  Inside the space editor, go to the **Space Settings** tab.

    ![Enable Group Chat](/images/community-hub/group-chat/enable-group-chat-1.webp)

4.  A popup will appear. Scroll Down and find the **Group Chat** option and turn the toggle switch to the "**On**" position.
5.  Click the **Save Chages** button to save your changes.

    ![Save Changes](/images/community-hub/group-chat/save-changes-2.webp)

## The Member Experience

Once Group Chat is enabled, members of that Space will see a new **Chat** tab in the Space navigation menu.

> **Note:** Make sure the FluentCommunity chat is enabled.

  ![Chat](/images/community-hub/group-chat/chat-3.webp)

Clicking on this tab will open the chat interface, where all members of the Space can send messages, emojis, and GIFs in real-time. This provides a dynamic and engaging layer to your community, separate from the more structured posts in the main feed.
