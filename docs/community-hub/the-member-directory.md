# The Member Directory

The Member Directory is a central hub where your users can discover, search for, and connect with other members of the community. It functions like an address book for your platform, making it easy for people to find each other.

> **Use Case:** A new member joins your community looking for experts on a specific topic. They can use the Member Directory to search for users who have "WordPress Developer" in their bio, helping them connect with the right people right away.

## Accessing the Member Directory

For members, the directory is accessible from the main community navigation bar. It is typically labeled **"Members."**

[Screenshot: The main community portal view with the "Members" tab highlighted in the top navigation.]

For administrators, you can view and manage members from the back-end by navigating to **FluentCommunity → Members** in your WordPress dashboard.

## Features of the Directory

The Member Directory is more than just a simple list. It includes several key features to help with discovery.

* **Search:** Members can use the search bar to find other users by name, username, or keywords from their profile bio.
* **Sorting & Filtering:** You can sort the directory by different criteria, such as "Last Active," "Newest Members," or "Alphabetical." This helps surface relevant profiles.
* **Profile Cards:** The directory displays members in a clean, card-based layout, showing key information at a glance, such as their profile picture, name, and a short bio or status.

[Screenshot: The front-end Member Directory page, showing the search bar, sorting options, and a grid of member profile cards.]
