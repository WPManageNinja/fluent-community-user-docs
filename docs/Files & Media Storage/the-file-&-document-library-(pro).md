# The File & Document Library (Pro)

The File & Document Library is a powerful feature that allows you to create a dedicated area for file sharing and resource management within any Space or Course. It functions like a built-in cloud storage system for your community.

> **Use Case:** In a "Project Collaboration" space, you can use the File Library to share project briefs, design mockups, and spreadsheets with the entire team. In a course, you can use it to provide a central library of all downloadable worksheets, e-books, and reference materials.

## Enabling the File Library

This feature can be enabled on a per-space or per-course basis.

1.  Navigate to the **Space** or **Course** where you want to add a library.
2.  In the editor, go to the **Settings** tab.
3.  Find the toggle switch labeled **"Enable File & Document Library"** and turn it on.

[Screenshot: The settings screen for a Space, with the "Enable File & Document Library" toggle switch highlighted.]

4.  Click **Update** to save your changes.

## How to Use the Library

Once enabled, a new **"Files"** tab will appear in the navigation menu for that Space or Course.

[Screenshot: The front-end view of a Space showing the new "Files" tab in its navigation.]

### Uploading Files

1.  Click on the **Files** tab to open the library.
2.  You can either **drag and drop** files directly onto the page or use the **Upload** button.
3.  You can upload various file types, including PDFs, documents, images, and ZIP files.

### Organizing with Folders

To keep your library organized, you can create folders.

1.  Click the **Create Folder** button.
2.  Give your folder a name and click **Create**.
3.  You can now drag and drop files directly into your folders.

### Restricting Visibility

For greater control, you can restrict who can see specific files or folders. This feature allows you to set permissions based on user roles (e.g., "Admins Only," "Moderators and Admins"), ensuring that sensitive documents are only accessible to the right people.

> **✨ Note:**
> The File & Document Library is separate from the standard media uploads in posts and comments. It is designed to be a more permanent and organized resource hub, whereas attachments to posts are part of the flowing conversation.
