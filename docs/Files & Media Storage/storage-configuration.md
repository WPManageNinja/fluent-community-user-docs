# Storage Configuration

This section covers how and where your community's media files are stored. As your community grows, managing file storage becomes essential for performance and scalability.

FluentCommunity provides several options, allowing you to start simple and scale up to a robust cloud-based solution when you're ready.

### What You Will Learn

In the following articles, we will explore the different storage configurations available:

* **Using the Local Server:** Learn about the default storage method that works out of the box.
* **Connecting Amazon S3 (Pro):** Offload your files to Amazon's powerful and scalable S3 service.
* **Connecting Cloudflare R2 (Pro):** Use Cloudflare's high-performance, S3-compatible storage with zero egress fees.
* **CDN & Image Proxy Setup (Pro):** Optimize file delivery speed for a global audience using a Content Delivery Network.
* **Secure File URLs (Pro):** Protect your premium content by generating temporary, signed links for private files.
