# Using The Quiz Module

Quizzes are an essential part of any online learning experience. They allow you to test your students' knowledge, reinforce key concepts, and track their comprehension of the course material.

> **Use Case:** At the end of a module on "Social Media Marketing," you can add a short quiz with multiple-choice questions to ensure students have understood the core strategies before they move on to the next section.

## How to Create a Quiz

Quizzes are added to your course structure just like lessons.

1.  Navigate to the course you want to add a quiz to.
2.  Go to the **Course Structure** or **Builder** tab.
3.  Under the appropriate section, click the **Add New Quiz** button.
4.  Enter a title for your quiz (e.g., "Module 1 Knowledge Check") and click **Add Quiz**.

This will create the quiz and take you to the quiz builder interface.

[Screenshot: The course builder view with the "Add New Quiz" button highlighted.]

## Building Your Quiz

The quiz builder is divided into two main areas: the quiz settings on the left and the question-building area on the right.

### Step 1: Add Your Questions

1.  In the main content area, click the **+ Add Question** button to create your first question.
2.  Write the question text in the field provided.
3.  Add your answer options. You can add multiple options and must select the radio button next to the **correct answer**.
4.  Repeat this process to add as many questions as you need.

[Screenshot: The quiz builder interface, showing a multiple-choice question being created with several answer options and the correct answer selected.]

### Step 2: Configure Quiz Settings

In the settings panel on the left, you can configure the behavior of your quiz.

* **Enable Comments:** Allow students to leave comments or questions on the quiz page.
* **Passing Grade:** Set the minimum score (as a percentage) that a student must achieve to pass the quiz.
* **Time Limit:** You can set a time limit (in minutes) for completing the quiz.
* **Quiz Retakes:** Define how many times a student can retake the quiz if they fail.

> **✨ Note:**
> Clearly defining the passing grade and retake policy in the quiz description is a great way to manage student expectations before they begin.

Once you have added your questions and configured the settings, click the **Update** button to save your quiz. It will now appear as part of your course structure for students.
