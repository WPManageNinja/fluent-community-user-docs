# Installation & Activation

Welcome to FluentCommunity! Let's get the plugin installed on your WordPress site so you can start building your community. This guide will walk you through the process for both the free and pro versions.

## Prerequisites

Before you begin, please ensure you have a self-hosted WordPress website up and running.

## Step 1: Installing the Plugin

You can install FluentCommunity directly from your WordPress dashboard or by uploading the plugin file if you have the Pro version.

### Option A: Installing the Free Version from WordPress.org

1.  From your WordPress dashboard, navigate to **Plugins → Add New**.
2.  In the search bar at the top right, type **"FluentCommunity"** and press Enter.
3.  You will see FluentCommunity in the search results. Click the **Install Now** button.

    [Screenshot: The WordPress plugin repository showing the FluentCommunity plugin with the "Install Now" button highlighted.]

4.  WordPress will automatically download and install the plugin.

### Option B: Installing the Pro Version via Upload

If you have purchased FluentCommunity Pro, you will have received a `.zip` file.

1.  From your WordPress dashboard, navigate to **Plugins → Add New**.
2.  Click the **Upload Plugin** button at the top of the page.
3.  Click **Choose File** and select the FluentCommunity Pro `.zip` file you downloaded.
4.  Click **Install Now** and wait for the installation to complete.

## Step 2: Activating the Plugin

After installation, you need to activate FluentCommunity:

1.  From your WordPress dashboard, navigate to **Plugins → Installed Plugins**.
2.  Find FluentCommunity in the list and click **Activate**.
3.  You will be automatically redirected to the FluentCommunity Setup Wizard.

Congratulations! FluentCommunity is now installed and activated on your site.
