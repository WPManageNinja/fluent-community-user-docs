# Student Tools

This section focuses on the features that directly impact your students' learning experience. These tools are designed to make your courses more engaging, accessible, and effective.

While Course Management deals with building the course structure, Student Tools are the features your learners will interact with directly.

### What You Will Learn

In the following articles, we will cover key features that enhance the student journey:

* **Enabling Lesson Previews:** Learn how to offer a "try before you buy" experience by making certain lessons free to non-enrolled students.
* **Using the Quiz Module:** Discover how to create and manage quizzes to assess knowledge and reinforce key concepts.
